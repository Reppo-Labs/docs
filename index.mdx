Reppo is a decentralized network for sourcing, curating, and monetizing AI training data. It is organized into domain-specific markets called datanets, where publishers contribute data and stake-backed voters curate quality. Both earn rewards for work the network finds valuable.

Whether you want to publish, vote, run a node, launch a datanet, or build programmatically, it starts here.
